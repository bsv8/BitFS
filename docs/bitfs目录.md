# 目录规范

## 跨操作系统
类型	Linux (XDG)	macOS	Windows
配置	~/.config/	 | ~/Library/Preferences/ | %APPDATA%
数据	~/.local/share/ | ~/Library/Application Support/ | %APPDATA% 或 %LOCALAPPDATA%
日志	~/.local/state/  | ~/Library/Logs/ | %APPDATA%\Logs\

## app name
main 网 app name 叫 bitfs
test 网 app name 叫 bitfs-test
